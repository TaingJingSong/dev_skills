# Style Compliance Checklist

> Check against the generated document to ensure InnoTech branding is correct.

## Branding
- [ ] "INNOTECH SOLUTION CO., LTD" spelled and cased exactly
- [ ] Tagline "Make Growth Simple with Innovative Technology" present on cover
- [ ] "Strictly Confidential" on every page footer
- [ ] No competing company logos or names present

## Colors (verify in docx XML if needed)
- [ ] Teal header fill = `1A949D` (not `2E75B6` or `0070C0` or any other blue)
- [ ] Light-blue label fill = `D9EAF7`
- [ ] Header text in teal cells = white `FFFFFF`
- [ ] Table borders = black `000000`
- [ ] Footer text color = `595959`

## Typography
- [ ] Body text: Cambria, 11pt (size: 22 in docx-js units)
- [ ] Heading 1: Cambria Bold, 14pt (size: 28)
- [ ] Heading 2: Cambria Bold, 12pt (size: 24)
- [ ] Table headers: Cambria Bold, 10pt (size: 20)
- [ ] Table body: Cambria, 10pt (size: 20)
- [ ] Footer: Cambria, 9pt (size: 18)
- [ ] No Times New Roman or Calibri present

## Layout
- [ ] Page size: A4 (11906 x 16838 DXA)
- [ ] All margins: 1 inch / 1440 DXA
- [ ] Cover page is in a separate section with no header/footer
- [ ] Body pages have consistent header and footer

## Tables
- [ ] `ShadingType.CLEAR` used (not SOLID)
- [ ] `WidthType.DXA` used (not PERCENTAGE)
- [ ] All tables have both `columnWidths` and per-cell `width`
- [ ] Cell padding applied: top/bottom 80, left/right 120 DXA
